Java Game Project: SandyMan

SandyMan is a game that was created by me. The inspiration behind this game was that I wanted to have a similar feel to PAC-Man because when I was a kid, I used to play PAC-Man a lot. This project consists of different classes, methods, Dynamic and Static bodies that are used to create the entire game.

I created several platforms for each level which has its own unique structure. Also, I created a Devil Character, Good Character, Coin Object and Fire Object. The only thing that changes throughout the level is that I changed the Devil Character and their position so that it wouldn't be the same as other levels. Finally, as you play the play and collect the coins, at the end of the game it shows the high score you've earned.
